Have a latching relay between each panel in Series. When open it breaks the circuit so the maximum voltage that can build up is equal to a single panel's open circuit voltage. otherwise an entire string of panels could build up very high voltages. Have a microcontroller with LoRa open the relays when sent a signal from headquarters Command & Control computer that is monitoring the weather radar. It opens them when there is a storm overhead and lightning risk. This should permanently fix the problem of solar arrays attracting lightning from a positive charge build up.

Microcontroller can serve multiple roles such as enabling micro inverters and rerouting the power in case of shade or other problems with the panel.
